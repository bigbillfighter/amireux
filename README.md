化妆品公司主页web网页前端设计，基于node.js+vue+element-ui实现

### 依赖
```
"axios": "0.21.1",
"element-ui": "2.15.2",
"pubsub-js": "1.9.3",
"vue": "2.5.2",
"vue-router": "3.0.1"
```

### 运行
> npm install // 安装依赖

> npm run dev // 启动
